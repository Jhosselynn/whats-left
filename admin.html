🔐 Seguridad

🔗 Conexión con Notion

Todos los datos se guardan en Notion y se leen desde ahí. Para que funcione en Netlify:

  1. Ve a notion.so/my-integrations → Crear integración → copia la key
  2. En Netlify → Site settings → Environment variables → agrega NOTION_KEY
  3. En Notion, abre cada base de datos → ··· → Connections → conecta tu integración
  4. Sube los archivos a Netlify (admin.html, whats-left.html, netlify.toml, netlify/functions/notion.js)
